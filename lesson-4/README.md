# Git branch

[Ссылка на документацию](https://www.atlassian.com/git/tutorials/using-branches)

1. Вывод всех веток

    ```bash 
    git branch
    ```

2. Переключится на ветку 

    ```bash
    git checkout name_branch
    ```

3. Создать ветку и переключится на нее

    ```bash
    git checkout -b name_branch
    ```
